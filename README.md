
起源理论（Origin Theory）

起源理论提出了一种结构假说，
试图解释文明形成和保持稳定所需要的最低条件。

Origin Theory proposes a structural hypothesis
to explain the minimal conditions required
for civilization to form and remain stable.

核心结构：
Origin = Boundary + Judgement + Correction
